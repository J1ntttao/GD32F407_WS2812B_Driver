## GD32F407VE 双SPI DMA驱动WS2812B

### 简介

本代码通过SPI0和SPI1两路接口，配合DMA驱动WS2812B LED灯带，实现高效的数据传输。

### 硬件要求

- MCU需支持SPI和DMA功能
- DMA需支持内存到外设传输

### SPI配置

- **时钟频率**：10.5MHz（WS2812B时序要求）
- **工作模式**：CPOL=0, CPHA=1
- **数据格式**：8位，MSB优先
- **MOSI**：数据输出
- **MISO**：未使用

### 注意事项

1. 确保SPI可配置到10.5MHz（参考芯片时钟树）
2. 空闲时SPI时钟保持低电平
3. 仅使用SPI发送功能

### 使用步骤

1. 配置系统时钟
2. 初始化SPI0和SPI1
3. 配置DMA通道
4. 连接WS2812B到SPI MOSI引脚
5. 调用显示函数

------

**提示**：实际应用前请测试时序是否符合WS2812B规格书要求。
